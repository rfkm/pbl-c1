pbl-c1-server
=============

## How to build

### Install deps

```bash
$ npm install
```

### Install gulp

```bash
$ npm install -g gulp
```

### Run build task

```bash
$ gulp
```

## How to run the server

```bash
$ gulp nodemon
```

## Other useful tasks

### watch

build + run + livereload
```bash
$ gulp watch
```
